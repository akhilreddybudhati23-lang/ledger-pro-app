# Ledger Pro - Retail POS System

A modern, easy-to-use point-of-sale (POS) system designed for retail stores, convenience stores, gas stations, and liquor stores.

## Features

### 👤 Employee Management
- Secure employee login with credentials
- Role-based access (Cashier, Manager, Admin)
- Shift clock-in and clock-out tracking
- Employee performance dashboard

### 💳 Point of Sale
- Quick product search and selection
- Add items to shopping cart
- Adjust quantities easily
- Multiple payment methods (Cash, Card, Check)
- Real-time sale processing
- Receipt generation

### 📦 Inventory Management
- Real-time inventory tracking
- Automatic inventory reduction on sale
- Low-stock alerts
- SKU and barcode support
- Inventory adjustment for damages or theft
- Stock level visibility

### 📊 Dashboard & Reporting
- Daily sales summary
- Sales by payment method (cash/card)
- Employee sales performance
- Transaction count and average transaction value
- Shift duration tracking
- Real-time data updates

### ⏰ Shift Management
- Employee shift tracking
- Clock in/out functionality
- Shift-based transaction isolation
- Cash reconciliation
- Discrepancy detection

## How to Use

1. **Login** - Enter your employee ID and password
2. **Search Products** - Use the search bar to find items
3. **Add to Cart** - Click products to add them to the shopping cart
4. **Adjust Quantities** - Use +/- buttons to change quantities
5. **Select Payment** - Choose payment method (Cash/Card/Check)
6. **Complete Sale** - Click "Complete Sale" button
7. **View Dashboard** - Check daily stats and performance
8. **View Inventory** - Monitor stock levels
